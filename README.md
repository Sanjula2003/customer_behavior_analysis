# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL, Power BI, and reporting tools. The objective is to extract insights from a dataset by performing data preparation, exploratory analysis, querying, visualization, and reporting.

The project covers the complete analytics pipeline, starting from loading and cleaning raw data to creating dashboards and presenting findings in a professional format.

## Tools & Technologies

The following tools and technologies were used throughout the project:

* **Python** – Data loading, cleaning, and analysis
* **Pandas & NumPy** – Data manipulation and preprocessing
* **Matplotlib & Seaborn** – Data visualization during EDA
* **PostgreSQL** – Database storage and SQL querying
* **Power BI** – Dashboard creation and business insights visualization
* **Gamma** – Presentation design and storytelling
* **Jupyter Notebook** – Development environment for analysis

## Project Workflow / Steps

### 1. Data Loading

* Imported the dataset into Python using Pandas
* Checked file structure and dataset dimensions
* Identified column names and data types

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distribution and summary statistics
* Identified trends, correlations, and patterns
* Created charts and visualizations for insights

### 3. Data Cleaning

* Removed duplicates
* Handled missing values
* Corrected inconsistent formatting
* Converted columns into suitable data types
* Prepared the dataset for analysis

### 4. SQL Analysis using PostgreSQL

* Loaded cleaned data into PostgreSQL Server
* Executed SQL queries to answer business questions
* Applied filtering, grouping, aggregation, and joins

### 5. Dashboard Development

* Connected Power BI to PostgreSQL or cleaned dataset
* Built interactive visual dashboards
* Included KPIs, trends, and category-level analysis

### 6. Reporting & Presentation

* Prepared a detailed project report
* Designed a professional PowerPoint presentation using Gamma
* Summarized business insights and recommendations

## How to Run

### Prerequisites

Install the following before running the project:

* Python 3.x
* PostgreSQL Server
* Power BI Desktop
* Jupyter Notebook

### Steps to Run

1. Clone the repository:

```bash
git clone https://github.com/Sanjula2003/customer_behavior_analysis.git
```

2. Navigate to the project folder:

```bash
cd project-name
```

3. Install required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn psycopg2
```

4. Open Jupyter Notebook and run the analysis notebook.

5. Import cleaned data into PostgreSQL.

6. Run SQL queries for analysis.

7. Open the Power BI dashboard file to explore insights.
